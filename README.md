### Objective
To calculate the mean and sum of NetSales grouped by Region using PROC MEANS.

### Dataset
Sample dataset stored in the WORK library.

### SAS Concepts Used
- PROC MEANS
- CLASS statement (group-wise analysis)
- VAR statement
- Statistical keywords (MEAN, SUM)
- Formatting using MAXDEC=

### Output
The program generates a summary dataset named `netsales_summary_by_region` containing
region-wise count of observations, mean NetSales, and total NetSales.

### Use Case
Commonly used in sales analysis and clinical data summarization.
